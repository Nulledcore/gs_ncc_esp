# gs_ncc_esp
Nullcore ESP for gamesense.pub

- Health Bar [gradient/flat]
- Name label
- Weapon label
- NCU label (based on steam64)
- Info panel (based on the old info panel on nullcore)
- Team based colors (none/spec/t/ct)
